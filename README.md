### **GitHub README: Bank Marketing Prediction**

> # Bank Marketing Term Deposit Classification
> 
> 
> ### Project Overview
> 
> 
> This project builds an end-to-end machine learning pipeline to predict whether a customer will subscribe to a bank term deposit based on direct marketing campaign data. The primary goal is to address class imbalance and optimize model thresholds for operational effectiveness.
> 
> 
> ### Key Methodologies
> 
> 
> * **Data Preprocessing:** Utilized `scikit-learn` `Pipeline` and `ColumnTransformer` to automate missing value imputation, standard scaling, and one-hot encoding.
> 
> 
> * **Imbalanced Classification:** Benchmarked four ensemble models (Logistic Regression, Random Forest, Gradient Boosting, and Balanced Bagging) using class weighting and resampling techniques.
> 
> 
> * **Metric Optimization:** Focused on AUC-PR and F1-score rather than accuracy to better reflect performance on the minority class.
> 
> 
> * **Threshold Analysis:** Performed custom decision thresholding to identify settings that achieve 90% recall, minimizing missed acquisition opportunities.
> 
> 
> 
>

> * **Libraries:** scikit-learn, Pandas, NumPy, Matplotlib
